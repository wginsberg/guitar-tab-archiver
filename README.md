# Guitar Tab Archiver

This extension used the [Plasmo](https://docs.plasmo.com/) framework.

## Build instructions
Built with
* Node 19.4.0
* pnpm 8.10.5
* MacOS 11.1 

To reproduce the build from source code perform the following steps:
```
pnpm install
pnpm build
```

This will generate a folder called `build/firefox-mv2-prod`
